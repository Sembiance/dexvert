## ✨ Features

- **3,672 formats supported** out of 7,686 known formats
- **662 helper programs** for comprehensive conversion
- Converts to modern formats: PNG, SVG, PDF, MP3, MP4, etc.
- Used by [discmaster.textfiles.com](http://discmaster.textfiles.com/) for legacy file conversion
- **36,032 sample files** available for testing across 4,061 formats

## 📋 Status

> ⚠️ **Note:** This project is not currently ready for public use. Development is ongoing.

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Sembiance/dexvert.git
cd dexvert

# Install dependencies (requires Python 3.x)
pip install -r requirements.txt
```

## 🚀 Usage

```bash
# Basic conversion
python dexvert.py input_file output_directory

# Example: Convert a legacy image format to PNG
python dexvert.py old_image.pcx ./output/
```
